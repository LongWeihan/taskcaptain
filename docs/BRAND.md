## Three core product points

### 1. It keeps running, not just replying

TaskCaptain does not stop at suggestions. It keeps the task moving until there is a result.

中文：
不是给建议就结束，而是持续推进，直到拿出结果。

### 2. It directs Codex automatically

It directs Codex step by step, manages execution flow, and keeps the work on course.

中文：
自动指挥 Codex 分步执行，管理流程，并让任务始终朝交付前进。

### 3. It keeps moving without supervision

Work keeps running while you sleep, switch context, or leave the keyboard.

中文：
即使你去忙别的、离开键盘或正在睡觉，任务也会继续推进。

## Brand voice

TaskCaptain should not sound like startup hype.
It should also not sound like a research paper.

Recommended voice traits:

### 1. Calm

Stay measured. Avoid shouting.

### 2. Commanding

Sound like something that takes responsibility and drives execution.

### 3. Outcome-driven

Keep attention on finished work, not abstract capability.

---

## Preferred vocabulary

Use more:
- run
- drive
- direct
- move
- deliver
- finish
- ship
- own

Use less:
- magical
- revolutionary
- next-generation
- intelligent assistant
- productivity booster

TaskCaptain should not feel like “a smart helper”.
It should feel like “someone actually responsible for getting the work done”.
